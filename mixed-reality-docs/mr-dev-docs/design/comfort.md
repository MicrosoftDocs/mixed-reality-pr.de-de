### <a name="best-practices-for-hololens-1st-gen-and-hololens-2"></a><span data-ttu-id="c6d71-136">Bewährte Methoden für HoloLens (1. Gen.) und HoloLens 2</span><span class="sxs-lookup"><span data-stu-id="c6d71-136">Best practices for HoloLens (1st gen) and HoloLens 2</span></span>
