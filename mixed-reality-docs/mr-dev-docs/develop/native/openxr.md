Windows Mixed Reality Headsets für Desktop-PCs.</span><span class="sxs-lookup"><span data-stu-id="08659-109">With OpenXR, you can build engines that target both holographic devices, like HoloLens 2, and immersive VR devices, like Windows Mixed Reality headsets for desktop PCs.</span></span> <span data-ttu-id="08659-110">Mit OpenXR können Sie Code schreiben, sobald dieser dann auf einer Vielzahl von Hardwareplattformen portierbar ist.</span><span class="sxs-lookup"><span data-stu-id="08659-110">OpenXR lets you write code once that's then portable across a wide range of hardware platforms.</span></span>

<span data-ttu-id="08659-117">Um den vollständigen Funktionssatz von HoloLens 2 als Ziel zu verwenden, verwenden Sie auch herstellerübergreifende und herstellerspezifische OpenXR-Erweiterungen, die zusätzliche Features über den OpenXR 1.0-Kern hinaus ermöglichen, z.
